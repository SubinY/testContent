# 小红书心理测评主题：隐藏人格类型  
**目标：同一主题 → 5种测评变体 → 便于A/B测试与批量生成**  
**补充：为降低AI幻觉 → 明确对应心理学理论基础与前置知识约束**

---

# ① 第一眼图像投射型（高点击停留）

## 玩法
用户观看一张**双关视觉图**并回答：  
> 第一眼看到什么？

## 内容结构
- 1张核心图像  
- 3–4种人格解释  
- 30秒内完成测试  
- 结果偏向：情绪、安全感、潜意识性格  

## 平台表现
- 点击率极高  
- 评论区讨论强  
- 易系列化更新  

## 心理学理论基础
- **投射测验理论**（Projective Tests）
- **罗夏墨迹测验原理（娱乐化改编）**
- **格式塔知觉理论**（Figure–Ground / 完形优先）
- **巴纳姆效应**（模糊描述的自我认同）

## AI前置知识约束
- 不声称具备真实诊断功能  
- 解释保持**概率性与象征性语言**  
- 结果必须为**非病理化描述**

## 可用API
- 文案生成：DeepSeek / OpenAI  
- 图像生成：Nano Banana  
- 二义性图像控制：多次采样 + 提示词约束

# ② 场景选择剧情型（高完播收藏）

## 玩法
沉浸式心理场景逐步选择：  
森林 → 房子 → 门 → 房间 → 遇见的人  

## 内容结构
- 4–6道剧情分支题  
- 输出完整人格画像  
- 偏成长叙事与自我探索  

## 平台表现
- 完播率高  
- 收藏率高  
- 适合连载  

## 心理学理论基础
- **荣格原型理论**（Archetypes）
- **叙事心理学**（Narrative Identity）
- **心理意象技术**（Guided Imagery）
- **象征主义投射机制**

## AI前置知识约束
- 场景象征需保持**跨文化通用性**  
- 不使用宗教/宿命式绝对结论  
- 结果解释遵循**成长导向而非定型标签**

## 可用API
- 剧情与结果生成：DeepSeek / OpenAI  
- 分支逻辑控制：Python规则引擎  
- 自动排版：Markdown模板

# ③ 情感依恋指数型（高转发变现）

## 玩法
10题选择题 → 输出：  
- 依恋类型  
- 恋爱风险  
- 适配伴侣画像  

## 内容结构
- 半量表结构（娱乐化）  
- 情绪共鸣强  
- 结果页较长  

## 平台表现
- 女性用户转发强  
- 评论互动高  
- 适合延伸付费报告  

## 心理学理论基础
- **依恋理论**（Bowlby / Ainsworth）
- **成人依恋模型**（安全型 / 回避型 / 焦虑型 / 混乱型）
- **亲密关系维度理论**（焦虑×回避）
- **情绪调节理论**

## AI前置知识约束
- 明确为**非临床评估**  
- 分数区间需参考**公开量表结构逻辑**  
- 建议部分必须**积极且可执行**

## 可用API
- 量表改写与结果文案：DeepSeek / OpenAI  
- 评分计算：Python权重模型  
- 长图生成：HTML → 图片渲染

# ④ 人生潜力预测型（标题爆款）

## 玩法
价值观选择 → 推断：  
- 隐藏天赋  
- 人生转折点  
- 幸福指数  

## 内容结构
- 5–7题选择  
- 强正向报告  
- 高情绪鼓励  

## 平台表现
- 点击率极高  
- 易做封面传播  
- 私域转化潜力强  

## 心理学理论基础
- **积极心理学**（Seligman）
- **优势理论**（Character Strengths）
- **自我效能感理论**（Bandura）
- **未来时间观理论**
- **巴纳姆效应（传播层面）**

## AI前置知识约束
- 禁止**确定性预言**  
- 使用**可能性/倾向性表达**  
- 所有结论需回归**个人努力与环境互动**

## 可用API
- 报告生成：DeepSeek / OpenAI  
- 批量生成：脚本循环调用API  
- 封面图生成：Nano Banana

# ⑤ 心理健康自评型（专业信任）

## 玩法
改编常见维度：  
- 焦虑  
- 情绪稳定度  
- 内耗水平  

输出：  
- 风险等级  
- 调节建议  
- 疗愈指引  

## 内容结构
- 9–12题  
- 语言温和  
- 强陪伴感  

## 平台表现
- 建立专业形象  
- 适合长期IP  
- 可导向咨询/课程  

## 心理学理论基础
- **认知行为理论（CBT）**
- **情绪评估量表结构**（PHQ-9 / GAD-7 逻辑参考，非直接使用）
- **压力-应对模型**（Lazarus）
- **正念理论**（Mindfulness）

## AI前置知识约束
- 必须加入**非医疗声明**  
- 不提供诊断或治疗承诺  
- 高风险提示需引导**现实求助资源**

## 可用API
- 专业化文案：DeepSeek / OpenAI  
- 个性化建议生成：OpenAI长文本  
- 语音疗愈扩展：TTS服务

# 自动化生成最小技术栈

## 文案层
- DeepSeek API  
- OpenAI API  

## 逻辑层
- Python评分规则  
- 模板化结果拼接  

## 图像层
- Nano Banana图像生成  
- HTML转图片用于长图发布  

## 生产方式
- 单主题批量生成5种测评形态  
- 每种形态多版本A/B测试  
- 基于数据反馈持续优化选题与结果结构
